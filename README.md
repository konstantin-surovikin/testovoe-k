# Тестовое задание

## Тредования

- Docker
- make

## Установка

Установка в Docker

```bash
sudo make install
```

Установка в Docker с тестовыми сидами

```bash
sudo make install-dev
```

Удаление

```bash
sudo make down-v
```

## API

API в формате Insomnia v4 в файле `API.json`
